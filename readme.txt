== Description ==

A framework to ease up the process of developing new plugins for wordpress. This plugin is therefore mainly for developers!

== Installation ==

To start using the uwebic framework just download & activate. You will then be able to access it's libraries and start creating your own plugins with the help of the uwebic framework. Read the documentation to find out how to get started.
